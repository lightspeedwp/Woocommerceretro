In wordpress block themes there are template parts, which are sections on a page that are possible shared with many pages or templates, examples are:

- Header
- Breadcrumbs
- Footer
- Mobile menu
- Sidebar

Each of these global shared elements are used sitewide in many instances. Patterns are registered to work with specific templates in WP block themes, which means you can use a consistent header, but on a checkout page you may want a distraction free header for example, so all you do is import the relevant distraction free header pattern into the global header template part.
