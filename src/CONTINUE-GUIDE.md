**Copy-paste this into your AI assistant:**

```
Continue PlayPocket project.
Date: [INSERT TODAY'S DATE - YYYY-MM-DD]

```

**That's it!** The AI will:
1. Read the master task list
2. Identify the next priority
3. Start working on it

## 💡 Pro Tips

### 1. Always Include the Date
```
Date: 2026-03-20
```
The AI uses this for file naming and reports.

### 2. Reference the Task List
```
Read /tasks/task-list.md
```
This ensures work on the right priority.

### 3. Ask for Summary When Uncertain
```
Summarize current status before proceeding.
```
Useful when returning after time away.

### 4. Be Specific When Needed
```
Work on: T4.3 - ShippingAddressForm.md
Reference: /tasks/blocks-guidelines-gaps.md
```

### 5. Request Confirmation for Big Changes
```
Wait for my confirmation before making changes.
```
Useful for risky or large modifications.
