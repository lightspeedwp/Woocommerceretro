### P2: Medium

- [ ] **A11Y2** — `text-neon-*` utility classes used as color overrides (25+ occurrences)
  - Files: 8 order block components (`AccountCreation.tsx`, `AdditionalFields.tsx`, `AdditionalInformation.tsx`, `AddressDetails.tsx`, `DownloadsSection.tsx`, `OrderDetails.tsx`, `OrderStatus.tsx`, `OrderStatusHeader.tsx`)

  - Action: Replace with BEM modifier classes (e.g., `.wp-order-detail__label--accent`)

- [ ] **A11Y3** — Missing focus management verification in key overlays

  - Action: Verify focus trapping works at runtime; add `FocusTrap` or `inert` if needed

  - File: `/src/app/components/blocks/order/OrderSummary.tsx` line 29
  - Issue: `md:wp-grid-cols-5` is a Tailwind-style breakpoint prefix pattern
  - Action: Convert to proper CSS media query in stylesheet
