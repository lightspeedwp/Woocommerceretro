# Store Notices

**WooCommerce Block:** `woocommerce/store-notices`
**Category:** `woocommerce`
**React Component:** Toast notification system
**File Location:** Various (uses toast utility)

---

## Block Definition

- **Name:** `woocommerce/store-notices`
- **Description:** Display shopper-facing notifications generated by WooCommerce or extensions.
- **Supports:**
  - `align` (full, wide)
  - `interactivity` (clientNavigation)
  - `multiple`
- **Attributes:** `align`

---

---

## Notice Types

| Type | Use Case | Color Token |
|------|----------|-------------|
| `info` | General information | `--wp--preset--color--primary` |
| `success` | Item added to cart, order placed | `--wp--preset--color--success` |
| `error` | Validation errors, payment failures | `--wp--preset--color--error` |
| `warning` | Low stock, limited availability | `--wp--preset--color--warning` |

---

## React Component Mapping

In the prototype, store notices are implemented via a toast notification system rather than inline blocks:

```tsx
// After adding to cart
toast.success('Product added to cart');

// Validation error
toast.error('Please select a size');

// Low stock warning
toast.warning('Only 3 left in stock');
```
