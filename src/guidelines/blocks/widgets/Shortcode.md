# Shortcode Block

**Type:** Block  
**Category:** Widgets  
**Status:** Complete  
**Last Updated:** 2026-02-03

---

## 📋 Overview

The Shortcode block allows you to insert WordPress shortcodes into the editor. It is primarily a backward-compatibility tool for plugins that haven't migrated to blocks yet.

---

## 🎯 Purpose

Use the Shortcode block to:
- Render legacy plugin shortcodes (e.g., `[contact-form-7]`).
- Insert dynamic content generated by PHP functions via shortcode API.

---

## 📐 Structure

```
Shortcode (div)
└── Dynamic Content
```

---

## 💻 Implementation

### **Props/Parameters**

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `text` | `string` | Yes | - | Shortcode string (e.g., `[gallery id="123"]`) |
| `className` | `string` | No | `''` | Additional CSS classes |

### **Code Example**

```tsx
import { Shortcode } from '@/components/blocks/widgets/Shortcode';

<Shortcode text="[my_shortcode]" />
```
