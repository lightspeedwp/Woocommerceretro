# InstagramFeed Pattern

**Version:** 1.0
**Type:** Pattern (Social Media)
**File:** `/src/app/components/patterns/InstagramFeed.tsx`
**BEM:** `.wp-instagram-feed__*`

---

## Overview

Displays a grid of Instagram-style images with optional overlay on hover showing likes/comments. Used for social proof on homepage or about pages.

---

## Retro / funky spec

- Glow image grid borders on hover
- Neon overlay with like/comment counts
- Glass overlay panel
- Spring hover scale

---

## Accessibility

- `aria-label="Instagram feed"` on container
- Image alt text describes content
- Overlay content accessible to screen readers
