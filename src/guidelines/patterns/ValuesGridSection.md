# ValuesGridSection Pattern

**Version:** 1.0
**Type:** Pattern (Brand Values)
**File:** `/src/app/components/patterns/ValuesGridSection.tsx`
**BEM:** `.wp-values-grid__*`

---

## Overview

Displays brand/company values in a card grid with icon, title, and description. Used on about and sustainability pages.

---

## Retro / funky spec

- Glow cards on hover
- Icon gradient circles (icon inside gradient-bordered circle)
- Glass card background
- Spring hover lift

---

## Accessibility

- `<section aria-label="Our values">` wrapper
- Icons are decorative: `aria-hidden="true"`
